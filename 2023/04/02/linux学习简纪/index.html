2 打开Linux命令行窗口

快捷键Ctrl + Alt + T,对应可以

  1. cd /home 打开路径对应文件夹

    可以用pwd观察现在所在的位置，用ll观察该路径下所有文件

  2. mkdir learn_linux 创建新的文件夹

  3. cd learn_linux 进入到创建的文件夹

  4. touch a 创建文件a（不要带拓展名）

    与windows不同，Linux文件一般不带后缀，（可以理解文件内容存放了文件的属性）

  5. cat a 查看a中内容（由于此时文件没有任何内容，故实际上没有什么反应）

  6. vi a 以命令模式进入文件

  7. 指令 作用
    i 进入输入模式，此时对文件进行正常的输入修改
    q 退出输入模式，进入命令模式，移动游标进行修改
    或者在底线命令模式中，在输入w保存文件后，输入q退出
    : 一般是对文件进行最后的保存退出操作

  8. cat a查看文件内容

  9. cp /home/lkx/linux_learn/a /home/lkx 将指定文件复制到指定位置

    cp OPS OPD 即cp 源文件 目的路径

  10. mv /home/lkx/linux_learn/a /home/lkx移动指定文件到指定位置

  11. 返回桌面，ll查看桌面文件

  12. rm 文件名 删除文件

  13. rm -r

    -r 从该文件夹向下递归，不管多少级目录，逐一删除

    -f 强行删除，没有提示

    -i 删除前逐一询问确定

    rm -r * 删除当前目录下所有文件和目录
